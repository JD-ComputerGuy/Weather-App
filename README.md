# Weather App 

A Python weather application that evolves from basic condition-based logic into a real API-powered tool using live weather data.

---

##  Features
- Search weather by city
- Live weather data via OpenWeather API
- Temperature and condition display
- Error handling for invalid input
- Favorites system (optional version)

---

##  Project Stages
- v1: Mock weather data  
- v2: Input-based logic  
- v3: API integration  
- v4: Output improvements + error handling  
- v5: Full CLI weather app  

---

##  Technologies
- Python 3
- Requests library
- OpenWeather API
- JSON data parsing

---

##  Future Improvements
- 5-day forecast feature
- GUI dashboard
- Web app version using Flask

---

##  Purpose
This project demonstrates API integration, data parsing, and building real-world Python applications.
